# ACME Meetings (React)

This is a minimal React frontend scaffold (Vite) with components matching the requested folder structure. Install and run:

```bash
cd acme-meetings-react
npm install
npm run dev
```

The project expects a backend API at `/api/...`. A simple Express + Mongoose server is included in `server/`.
# Acme Meetings (React + Express + MongoDB)

Front-end: React (Vite). Backend: Express + Mongoose. Replace placeholder MongoDB URI with your string in `server/.env`.

Quick start:

1. Install front-end deps:

```bash
cd acme-meetings-react
npm install
npm run dev
```

2. Start backend API:

```bash
cd acme-meetings-react/server
npm install
npm run dev
```
